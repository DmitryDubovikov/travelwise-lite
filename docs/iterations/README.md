# docs/iterations — как читать эти папки

Каждая закрытая итерация оставляет тут папку `NN/` (двузначный номер) с тремя
файлами. Формат общий для всей семьи -lite-проектов:

- **`explainer.md`** — что итерация сделала и зачем, простым языком. Начинается
  с продуктовой ценности, затем TL;DR, ликбез по новым техникам через
  сиблинг-аналоги (ADK-концепции объясняются через знакомый LangGraph/CrewAI),
  обязательная секция «⚖️ Специфика ADK», поток данных с псевдографикой и карта
  «где в коде» с номерами строк на момент итерации.
- **`demo.md`** — happy-path смок: пронумерованные копипаст-исполнимые шаги,
  под каждым — ожидаемый результат. Прогоняется вживую при закрытии итерации.
  Шаги с пометкой «⚠️ live» зовут Gemini и жгут free-tier RPM.
- **`learnings.md`** — почему решения приняты именно так, грабли (особенно «где
  ADK повёл себя не как LangGraph/CrewAI»), сознательные срезы и отложенные TODO.

Итерации: см. `ROADMAP.md` в корне — там статусы и скоупы. Спеки, по которым
итерации строились, лежат в `specs/NN/`.
